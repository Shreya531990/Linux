1. install package
#yum install podman container-tools -y
--------------------------------------------------------
2. check podman version
#podman  -v
or
#rpm  -q podman
-------------------------------------------------------
3. check container images
#podman images
-------------------------------------------------------
4. show running container
#podman ps
-------------------------------------------------------
5. show running & stopped container
#podman  ps  -a
-------------------------------------------------------
6. for pull images of ubuntu
#podman  pull  ubuntu
-------------------------------------------------------
7. run container with -dt mode
#podman  run  -dt  ubuntu

-d   detach
-t   terminal
-------------------------------------------------------
8. run container with -it mode
#podman run  -it  ubuntu  /bin/bash

-i   interact
-t   terminal

Note:
after exit command -it mode container will be stopped, -dt mode container still running
-------------------------------------------------------
9. to go inside -it mode container
#podman   attach  <container name/ID>
-------------------------------------------------------
10. to go inside -dt mode container
#podman   exec   -it   <container name/ID>  /bin/bash
-------------------------------------------------------
11. for stop running container
#podman   stop   <container name/ID>
-------------------------------------------------------
12. for kill running container
#podman   kill   <container name/ID>
-------------------------------------------------------
13. for start stopped container
#podman   start   <container name/ID>
-------------------------------------------------------
14. for remove stopped container
#podman  rm   <container name/ID>
-------------------------------------------------------
15. for stop all running container
#podman  stop   -a
-------------------------------------------------------
16. for kill all container 
#podman  kill   -a
-------------------------------------------------------
17. for remove all stop container
#podman  rm   -a
-------------------------------------------------------
18. for provide name to container
#podman  run  -dt  --name=c1  ubuntu
-------------------------------------------------------
19. run httpd into container and expose port 81
#podman  run  -dt   --name=c2  -t  -p  81:80   httpd

for verify default web page
open web browser
http://192.168.0.104:81
-------------------------------------------------------
20. run httpd and host custom web page
#podman  run  -dt   --name=c3  -t  -p  82:80   httpd

for verify default web page
open web browser
http://192.168.0.104:82

to go inside container
#podman exec  -it  c3   /bin/bash
